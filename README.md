# 💫 About Me:
Estudiante de Ingeniería de Sistemas apasionado por el desarrollo de software, con conocimientos sólidos en programación, bases de datos, redes, arquitectura de sistemas y desarrollo de aplicaciones distribuidas. Tengo experiencia desarrollando proyectos full stack, aunque mi enfoque principal está en el backend, aplicando buenas prácticas, metodologías ágiles, APIs REST y despliegue de soluciones web en entornos distribuidos. También me interesa la computación en la nube, la ciberseguridad y el desarrollo de startups tecnológicas.
